Introduction

Base URL: http://mise-en-relation.com

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Endpoints

POST api/login

Example request:
curl --request POST \
    "http://mise-en-relation.com/api/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"login\": \"quidem\",
    \"password\": \"consequatur\"
}"

const url = new URL(
    "http://mise-en-relation.com/api/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "login": "quidem",
    "password": "consequatur"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

POST api/register

Example request:
curl --request POST \
    "http://mise-en-relation.com/api/register" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "lastName=et"\
    --form "firstName=at"\
    --form "phone=iste"\
    --form "email=joshua.nolan@example.net"\
    --form "password=<jdr55>aR0<C&ZGfH]2"\
    --form "role=reprehenderit"\
    --form "img=@C:\Users\FABRICE\AppData\Local\Temp\phpE2BD.tmp" 
const url = new URL(
    "http://mise-en-relation.com/api/register"
);

const headers = {
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('lastName', 'et');
body.append('firstName', 'at');
body.append('phone', 'iste');
body.append('email', 'joshua.nolan@example.net');
body.append('password', '<jdr55>aR0<C&ZGfH]2');
body.append('role', 'reprehenderit');
body.append('img', document.querySelector('input[name="img"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
    body,
}).then(response => response.json());
